List demo

This project represents the incremental learning steps required to learn to use
the MDD File System library. (Requires MAL1210 and XC16 compiler v1.1)


It will wait for an SD card to be inserted and then proceed to list the top 16
text files present on the card.
Selection is performed scrolling up and down the list by pointing to the top or
bottom of the screen and selection is performed by touching the right side of
screen.

The selected file is listed (first 500 characters at most)
After a prompt the process will be repeated
